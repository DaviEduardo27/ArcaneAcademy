# Plano Inicial da Versao Mobile

1. Copiar a base jogavel da `Web/` para `mobile/webview-build/` quando a versao web estiver estavel.
2. Adaptar a interface para controles por toque.
3. Rever tamanhos de botoes, textos e HUD para telas menores.
4. Revisar desempenho para telas mobile.
5. Reduzir peso de imagens, sons e videos se necessario.
6. Preparar o empacotamento futuro via WebView/Capacitor.
7. Integrar ads somente apos a estabilidade da versao mobile.
