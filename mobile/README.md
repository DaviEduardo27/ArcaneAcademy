# Arcane Academy Mobile

Esta pasta sera usada para a futura versao Android do jogo.

A versao web atual permanece isolada em `Web/`, sem mistura com a estrutura mobile nesta etapa.

A ideia desta base e adaptar o jogo atual para controles por toque, navegacao mobile e telas menores, preservando primeiro a estabilidade da versao web.

Anuncios serao implementados somente depois que a versao mobile estiver estavel.

A pasta `webview-build/` sera a base de adaptacao do jogo para o fluxo mobile.

A pasta `assets/` guardara recursos especificos da versao mobile ou arquivos otimizados para desempenho em dispositivos moveis.
